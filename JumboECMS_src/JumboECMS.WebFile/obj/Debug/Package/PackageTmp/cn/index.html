      浙江创源环境科技股份有限公司（原名浙江创源环境监控技术有限公司），是由嘉兴市环境科学研究所有限公司与嘉兴市创见智能科技有限公司于2005年共同出资成立的一家从事高科技环保产业的股份制有限公司，注册资金1200万元，年营业额3000万元以上。 公司专业从事污染源在线监测产品的开发、研制及污染源（水、气）在线监控工程的安装、维护。 [详细介绍....]
