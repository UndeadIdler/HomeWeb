  • 公司简介

　　 我公司全称为浙江创源环境科技股份有限公司（原名浙江创源环境监控技术有限公司），是由嘉兴市环境科学研究所有限公司与嘉兴市创见智能科技有限公司于2005年共同出资成立的一家从事高科技环保产业的股份制有限公司，注册资金1200万元，年营业额3000万元以上。 公司专业从事污染源在线监测产品的开发、研制及污染源（水、气）在线监控工程的安装、维护。

          在2007年度，为配合省环保局“811”系统工程的建设及实现“节能减排”的总体目标，公司参与了浙江省的在线监控工程并网建设工作，并使嘉兴地区在全省范围内率先完成了年度任务。同时，公司还选派骨干力量，积极参与省内其它地区的建设工作，从而得到了各地各级环保部门的高度肯定。2008年8月，公司获得了国家环保总局颁发的“环境污染治理设施运营资质证书（水、气）（正式）”；9月，公司当选浙江省环境监测协会理事单位。目前，公司专项承担了嘉兴地区污染源在线监控工程建设及运营业务。
