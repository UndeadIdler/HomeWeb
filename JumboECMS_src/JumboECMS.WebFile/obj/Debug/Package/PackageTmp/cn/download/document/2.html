V1.0操作手册

发布时间：2012-03-30  浏览量：  【字体：大 中 小】

　　1 网站的环境配置

　　1.1 环境要求

　　1、 操作系统：Windows Server2003(推荐)或Windows 2000 Server、Windows XP

　　2、 IIS7.0、IIS6.0

　　3、 .NET Frameworks2.0

　　4、 Sql Server 2005(推荐)或Sql Server 2000

　　1.2 环境配置(实例)

　　1、 将JumbotCms_www目录下所有拷贝到服务器下，如E:\wwwroot\jcms。

　　2、 打开IIS，选中默认站点(假设端口号为1104)，鼠标右键——>新建虚拟目录，输入别名jcms。

　　3、 指向CMS目录，E:\wwwroot\jcms

　　4、 只勾选“读取权限”，保存。

　　5、 此时虚拟目录状态如下

　　6、 再次鼠标右键点击虚拟目录，选择“属性”,创建应用程序名，并选择“纯脚本”执行权限。

　　7、 切换至“ASP.NET”，选择2.0.50727版本。

　　8、 再切换到“文档”，查看默认文档是否包含：default.aspx、index.htm，如果没有

　　则手工添加。点击确定后保存。

　　1.3 环境配置(扩展)

　　1.3.1 视频模块

　　如果网站有视频模块，并且使用.flv，那么就需要：通过IIS进行添加：单击【开始】→【程序】→【管理工具】→【IIS管理器】，逐步展开“本地计算机”、“网站”，在你的网站上右击，选择【属性】，单击“HTTP头”选项卡→单击“MIME类型”按钮，再单击“新建”按钮，在“扩展名”框内输入“.flv”，“MIME类型”框中输入“flv-application/octet-stream”

　　1.3.2 伪静态

　　如果网站需要使用非.aspx后缀的伪静态，如class-1-1.htm、class-1-2.abc等，那么：打开IIS-->右击默认网站-->属性-->主目录-->找到配置按钮-->映射-->点击“插入”-->可执行文件:c:\windows\microsoft.net\framework\v2.0.50727\aspnet_isapi.dll ,去掉检查文件是否存在”复选框的钩-->保存即可。

　　1.4 CMS系统安装

　　将博CMS的前台发布有多种选择，分别是动态化、html静态化和shtml静态化，拥有自己服务器的朋友可选择shtml模式，因为shtml可支持局部更新，从而在保证页面数据更新的情况下，减缓了服务器消耗，设置此项可打开\_data\config\site.config……

　　1、 打开IE，进入：http://127.0.0.1:1104/jcms/install/，进入安装, 前2个页面大家自己看一下，我就不叙述了;(如果需要重复安装，请先重启IIS服务)

　　2、 到第3页，我们以Sql Server数据库为例，输入已有的数据库的信息，如果没有就先建立一个数据库;

　　3、 输入完信息后，开始安装，系统自动创建数据表和相关的记录;

　　4、 大概2-4秒之后出现如下界面，此时可以输入您网站的前后台用户登录信息后，点击“确定保存”;

　　5、 此时cms安装结束了，现在您可以开始建站之旅了，记得删除install目录。

　　6、 输入正确的前台用户信息后，点击“管理中心”;

　　7、 输入正确的后台用户信息后，点击“登录”，进入后台管理中心。
